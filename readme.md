# ships

> Two player space(?) ship dogfighting game

## Tech

- This is a [Lua](https://lua.org/) code base, built on top of the [LÃ–VE](https://love2d.org/) game development framework

## Prerequisites

Install Love on your machine, preferably with [Homebrew](https://formulae.brew.sh/cask/love):

```bash
$ brew install --cask love
```

## Run

Clone this repository, then

```bash
$ cd gravity
$ love .
```
